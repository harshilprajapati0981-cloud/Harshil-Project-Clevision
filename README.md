# Clevision

A responsive landing page built as part of a frontend development assessment. The project demonstrates modern UI design, responsive layouts, and JavaScript functionality.

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6+)
* Tailwind CSS
* Git & GitHub

## Features

* Responsive design
* Hero section
* Product filtering and grouping challenges
* Contact section
* Modern UI using Tailwind CSS

## How to Run Locally

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder

```bash
cd Clevision
```

3. Install dependencies

```bash
npm install
```

4. Start Tailwind build

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

5. Open `index.html` in your browser.
